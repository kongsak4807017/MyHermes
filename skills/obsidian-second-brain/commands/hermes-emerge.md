# /hermes-emerge

Surface unnamed patterns from recent notes.

## Steps

1. Read `_HERMES.md` first
2. Load L0-L2 context
3. Scan last 30 days of notes:
   - Daily notes
   - Project updates
   - Logs
   - Ideas
4. Identify unnamed patterns:
   - Same problem mentioned across contexts
   - Recurring themes without labels
   - Connected ideas not yet linked
   - Emotional patterns (stress, excitement)
5. Name each pattern
6. Create or update concept notes
7. Link source notes
8. Report: patterns surfaced, actions suggested

## Example

```
Patterns surfaced:
1. "Onboarding friction"
   Found in: 4 daily notes, 2 project updates
   You never named it, but it's your bottleneck.
   → Created: wiki/concepts/Onboarding Friction.md

2. "Decision fatigue at 3 PM"
   Found in: 5 daily notes mention low energy afternoons
   → Suggested: Batch decisions to mornings
```
