# Slow Local Model Batch Processing

Use this skill when setting up batch jobs, cron jobs, or overnight processing with slow local LLMs (e.g., Ollama models that take 3-10 minutes per response). Covers WSL venv setup, Hermes cron integration, bash wrapper patterns, and TTS-ready output generation.

## When to Use This Skill

- Model responds in 3+ minutes per query
- Running overnight while user is asleep
- Generating long-form content (2,500+ chars) in batches
- Using local models via ollama-local, llama.cpp, or similar
- Need TTS-ready output (no brackets, no stage directions)
- Running in WSL with Hermes Agent's Windows venv

## Architecture Pattern

```
[Cron Trigger] → [Bash Wrapper] → [WSL Venv Python] → [Hermes AIAgent]
                                                    ↓
                                              [Local LLM via Ollama]
                                                    ↓
                                              [Post-process + Save]
```

**Expected:** Response within 30-60 seconds for short prompts.

### Phase 3: Create Python Script

```python
#!/usr/bin/env python3
"""Batch narrative generator — runs with slow local models."""

import sys
from pathlib import Path

# Hermes Agent lives on Windows drive
HERMES_AGENT = Path("/mnt/c/Users/User/hermes-agent")
sys.path.insert(0, str(HERMES_AGENT))

from run_agent import AIAgent

# Check deps
try:
    from dotenv import load_dotenv
except ImportError:
    print("ERROR: Run with /tmp/hermes-run-venv/bin/python")
    sys.exit(1)

OUTPUT_DIR = Path.home() / ".hermes" / "narratives" / "news"
OUTPUT_DIR.mkdir(parents=True, exist_ok=True)

def generate_with_agent(prompt, provider="ollama-local", model="gemma4:e4b"):
    agent = AIAgent(
        provider=provider,
        model=model,
        max_iterations=2,      # Limit to prevent runaway
        quiet_mode=True,
    )
    return agent.chat(prompt)

def clean_for_tts(text):
    """Remove stage directions and meta-text for TTS."""
    import re
    text = re.sub(r'\[.*?\]', '', text)
    text = re.sub(r'\(.*?\)', '', text)
    text = re.sub(r'#{1,6}\s*คำแนะนำ.*?(?=---|$)', '', text, flags=re.DOTALL)
    text = re.sub(r'\*\*💡.*?(?=---|$)', '', text, flags=re.DOTALL)
    text = re.sub(r'\[SCRIPT\]\s*', '', text)
    return text.strip()

def save_narrative(content, style_name, date_str):
    filename = f"{date_str}_{style_name.replace(' ', '_')}.md"
    filepath = OUTPUT_DIR / filename
    md = f"""# Narrative — {style_name} ({date_str})

{content}

---
*Generated by {model} via {provider}*
"""
    filepath.write_text(md, encoding='utf-8')
    return filepath

def main():
    from datetime import datetime
    date_str = datetime.now().strftime('%Y-%m-%d')
    
    # Your prompt templates here
    prompts = {
        "Standard News": "...",
        "Friend-to-Friend": "...",
        "Expert Deep Dive": "...",
    }
    
    for style, prompt in prompts.items():
        print(f"Generating {style}...")
        raw = generate_with_agent(prompt)
        clean = clean_for_tts(raw)
        filepath = save_narrative(clean, style, date_str)
        print(f"Saved: {filepath} ({len(clean)} chars)")

if __name__ == "__main__":
    main()
```

### Phase 4: Create Bash Wrapper

```bash
#!/bin/bash
# ~/.hermes/scripts/run_nightly_batch.sh
VENV_PYTHON="/tmp/hermes-run-venv/bin/python"
SCRIPT="/home/user/.hermes/scripts/nightly_batch.py"

cd /mnt/c/Users/User/hermes-agent || exit 1
exec "$VENV_PYTHON" "$SCRIPT" "$@"
```

Make executable:
```bash
chmod +x ~/.hermes/scripts/run_nightly_batch.sh
```

### Phase 5: Register Cron Job

```bash
hermes cron create "nightly-batch" \
  --schedule "0 2 * * *" \
  --script "run_nightly_batch.sh"
```

**Important:** Script name must be relative to `~/.hermes/scripts/`, not absolute path.

**Verify:**
```bash
hermes cron list
```

## TTS-Ready Prompt Engineering

### Constraints for Spoken Output

```
CRITICAL RULES:
1. ห้ามมีคำนำหรือคำลงท้าย: เริ่มรันเนื้อหา [SCRIPT] ทันที
2. บทพูดภาษาพูดล้วนๆ (PURE SPOKEN LANGUAGE)
3. ห้ามใส่วงเล็บและเลขอ้างอิง
4. ห้ามใส่คำแนะนำสำหรับคนพากย์ ห้ามใส่ Sound Effect
5. ความยาว: ประมาณ 2500-2700 ตัวอักษร
```

### Post-Processing (Always Required)

Local models often ignore constraints. Always run post-processing:

```python
import re

def clean_for_tts(text):
    # Remove stage directions [text] and (text)
    text = re.sub(r'\[.*?\]', '', text)
    text = re.sub(r'\(.*?\)', '', text)
    
    # Remove Thai meta-sections
    text = re.sub(r'#{1,6}\s*คำแนะนำ.*?(?=---|$)', '', text, flags=re.DOTALL)
    text = re.sub(r'\*\*💡.*?(?=---|$)', '', text, flags=re.DOTALL)
    text = re.sub(r'#{1,6}\s*ข้อแนะนำ.*?(?=---|$)', '', text, flags=re.DOTALL)
    
    # Remove SCRIPT markers
    text = re.sub(r'\[SCRIPT\]\s*', '', text)
    text = re.sub(r'\[/?SCRIPT\]', '', text)
    
    # Remove markdown tables (TTS can't read them well)
    text = re.sub(r'\|[^\n]+\|', '', text)
    
    return text.strip()
```

## Pitfalls

### 1. Windows Venv from WSL
**Symptom:** `python.exe: can't open file 'C:\home\user\...'`
**Fix:** Create separate WSL venv at `/tmp/hermes-run-venv/`

### 2. PEP 668 Externally Managed
**Symptom:** `error: externally-managed-environment`
**Fix:** Always use `python3 -m venv`, never `pip install` directly on system Python

### 3. Cron Job Script Path
**Symptom:** `Script path must be relative to ~/.hermes/scripts/`
**Fix:** Use filename only, not absolute path. Create bash wrapper if needed.

### 4. Background Process Hang
**Symptom:** No output for minutes, process appears stuck
**Fix:** Use `process(action="kill")` to stop (exit code -15 = SIGTERM). Check if model is loading or network is slow.

### 5. Tool Loop Warning
**Symptom:** `[Tool loop warning: same_tool_failure_warning; count=3]`
**Fix:** Switch to different tool or approach. Don't retry same failing command.

### 6. Meta-Text in Output
**Symptom:** Model produces "คำแนะนำในการนำไปใช้" despite constraints
**Fix:** Always run post-processing. Constraints alone are not enough.

### 7. Ollama Not Running
**Symptom:** `curl: (7) Failed to connect to localhost:11434`
**Fix:** Start Ollama: `ollama serve` or check if it's running on different host

### 8. Kimi API 404 / Wrong Endpoint (CRITICAL — May 2026)
**Symptom:** `NotFoundError [HTTP 404]` at `https://api.kimi.com/coding`
**Root cause:** Hermes Agent's `kimi-coding` provider maps to `api.kimi.com/coding` which doesn't support chat completions. The Kimi For Coding CLI uses a different auth flow.
**Fix:** Do NOT use `provider="kimi"` or `provider="moonshot"` with `model="kimi-for-coding"` for API calls. Instead:
- Use `provider="kimi-coding"` with correct API key from `KIMI_CODE_API_KEY`
- Or use a different fast model (e.g., another Ollama model, OpenAI, Anthropic)
- Or fetch news via web scraping / RSS directly without LLM

**Verified non-working combinations (May 2026):**
```python
# FAILS — 404 Not Found
AIAgent(provider="kimi", model="kimi-for-coding")
AIAgent(provider="moonshot", model="kimi-for-coding")
AIAgent(provider="kimi-coding", model="kimi-for-coding")  # Hangs/timeout
```

**Working alternatives for fast fetch:**
- `provider="openrouter"` with any fast model
- `provider="anthropic"` with Claude
- `provider="openai"` with GPT-4o-mini
- Direct RSS/NewsAPI without LLM
- Another Ollama model (e.g., phi4, llama3.2) if available locally

### 9. Model Tries to Use Nonexistent Tools
**Symptom:** `Unknown tool 'google:search' — sending error to model for self-correction`
**Fix:** Set `max_iterations=2` and `quiet_mode=True` on AIAgent. The model will self-correct after the error. If it persists, disable toolsets entirely: `enabled_toolsets=[]`.

### 10. SearXNG Blocked by Search Engines
**Symptom:** `SearxEngineTooManyRequestsException: Too many request (suspended_time=180)`
**Fix:** SearXNG scrapes search engines and gets rate-limited. Solutions:
- Use engines that don't block (DuckDuckGo, Qwant, Brave)
- Add delay between requests
- Use dedicated news APIs (NewsAPI, GNews) instead

## Hybrid Architecture: Fast Fetch + Slow Generate

When the slow local model is also needed for content generation (not just processing), use a two-stage pipeline:

```
Stage 1: Fast Model (Kimi/OpenAI/Claude/another Ollama model)
  └─> Fetches news, summarizes, builds context
        ↓
Stage 2: Slow Local Model (gemma4:e4b, llama3, etc.)
  └─> Generates long-form narrative from context
        ↓
Post-Process: Clean for TTS
  └─> Save to ~/.hermes/narratives/news/
```

### Stage 1: News Fetch Options (Ranked by Reliability)

| Method | Speed | Reliability | Setup |
|--------|-------|-------------|-------|
| Mock data (testing) | Instant | 100% | None |
| RSS feeds (direct) | Fast | High | `feedparser` |
| NewsAPI (free tier) | Fast | High | API key |
| Web scraping (requests+BS4) | Medium | Medium | None |
| SearXNG (self-hosted) | Medium | Low (blocked) | Install + config |
| Kimi API | Fast | Low (404 errors) | KIMI_CODE_API_KEY |

**Recommendation:** For production cron jobs, use RSS feeds + NewsAPI as primary, with mock data fallback.

### Stage 2: Slow Model Generation

Always use `max_iterations=2` to prevent runaway tool loops:

```python
gemma_agent = AIAgent(
    provider="ollama-local",
    model="gemma4:e4b",
    max_iterations=2,        # CRITICAL: prevents 10+ min loops
    quiet_mode=True,
    enabled_toolsets=[],       # Optional: disable tools entirely
)
```

### Full Hybrid Script Example

```python
#!/usr/bin/env python3
"""Hybrid: Fast fetch (RSS/NewsAPI) + Slow generate (gemma4:e4b)"""

import sys
from pathlib import Path

HERMES_AGENT = Path("/mnt/c/Users/User/hermes-agent")
sys.path.insert(0, str(HERMES_AGENT))

from run_agent import AIAgent

def fetch_news_rss():
    """Fast news fetch via RSS — no API key needed"""
    import feedparser
    feeds = [
        "https://www.bangkokpost.com/rss/topstories.xml",
        "https://www.thairath.co.th/rss/news",
    ]
    items = []
    for url in feeds:
        try:
            d = feedparser.parse(url)
            for entry in d.entries[:3]:
                items.append(f"[{entry.title}] {entry.get('summary', '')[:200]}")
        except Exception as e:
            print(f"RSS error: {e}")
    return "\n".join(items) if items else None

def fetch_news_mock():
    """Fallback mock data for testing"""
    return """[1] ดาราสาวชื่อดังโพสต์ภาพคู่กับนักธุรกิจหนุ่ม...
[2] รัฐบาลประกาศมาตรการใหม่เรื่องภาษีดิจิทัล...
[3] นักฟุตบอลทีมชาติไทยยิงประตูชัยนาทีสุดท้าย..."""

def generate_narrative(news_content, style_prompt, agent):
    prompt = style_prompt.format(news_content=news_content)
    raw = agent.chat(prompt)
    return clean_for_tts(raw)

def main():
    # Stage 1: Fast fetch
    news = fetch_news_rss() or fetch_news_mock()
    
    # Stage 2: Slow generate
    agent = AIAgent(provider="ollama-local", model="gemma4:e4b",
                    max_iterations=2, quiet_mode=True)
    
    for style_name, template in STYLES.items():
        content = generate_narrative(news, template, agent)
        save_narrative(content, style_name)

if __name__ == "__main__":
    main()
```
